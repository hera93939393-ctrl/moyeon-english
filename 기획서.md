# 모연 English — 기획 문서

## 서비스 개요
**모연 English**는 초등·중등 영어 학원의 안내 챗봇 겸 레벨테스트 예약 웹 서비스입니다.
학부모는 전화 없이 학원 정보(커리큘럼·시간표·셔틀·수강료·할인 등)를 챗봇으로 확인하고,
레벨테스트를 직접 예약·변경·취소할 수 있습니다. 원장은 예약이 들어올 때마다 문자로
즉시 알림을 받고, 관리자 화면에서 정원 현황과 예약 목록을 확인합니다.

## 해결하는 문제
학원에 대해 궁금한 학부모는 매번 전화로 문의해야 하고, 레벨테스트 예약도 통화로
일정을 조율해야 해 번거롭습니다. 학원 입장에서도 같은 문의에 반복 응대해야 하고,
예약 인원이 몰릴 때 정원을 수기로 관리하기 어렵습니다. 본 서비스는 이 과정을
웹 하나로 자동화합니다.

## 타겟 사용자
| 사용자 | 니즈 |
|---|---|
| 학부모 | 전화 없이 정보 확인, 레벨테스트 예약·변경·취소 |
| 원장(운영자) | 예약 문자 알림 즉시 수신, 정원·예약 현황 관리 |

## 핵심 기능
- 정보 안내 챗봇 (버튼 클릭 + 자유 질문 → 항목별 업로드 자료 검색 기반 답변)
- 관리자 페이지에서 항목별 자료 업로드 (커리큘럼·레벨테스트·시간표·셔틀버스·수강료·초등·중등·할인 8개 항목 각각에 PDF/텍스트 파일 등록·교체·삭제, 등록된 항목은 그 자료로 챗봇이 답변)
- 레벨테스트 예약 (달력 UI, 정원 관리: 일 30명·시간대별 2명)
- 예약 변경·취소 (본인 인증 토큰 방식)
- 원장 문자 알림 (SOLAPI 연동)
- 관리자 화면 (`/admin.html` — 카테고리별 자료 업로드, 예약·정원 현황. 개인정보 보호를 위해 로그인 계정은 비공개이며, 동작은 시연 영상 참고)
- 개인정보 30일 자동 삭제

## 기술 스택
- **프론트엔드**: HTML/CSS/JavaScript
- **백엔드**: Node.js + Express
- **DB**: PostgreSQL (Neon, 클라우드) — 로컬 개발 시 SQLite 자동 대체
- **문자 알림**: SOLAPI SMS API
- **배포**: Railway (GitHub 연동, 자동 배포)
